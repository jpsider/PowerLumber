# New-Log

## SYNOPSIS
Clears logs in a directory older than the specified number of days.

## SYNTAX

```
New-Log [-Logfile] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Clears logs in a directory older than the specified number of days.

## EXAMPLES

### EXAMPLE 1
```
New-Log -Logfile c:\temp\new.log
```
